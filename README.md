# Weather Forecast Service
Сервис для получения прогноза погоды с кэшированием в Redis и визуализацией данных.

## Описание
- Сервис предоставляет прогноз погоды на 24 часа для любого города:
- Получает координаты города через Geocoding API
- Запрашивает погоду через Open-Meteo API
- Кэширует данные в Redis на 15 минут
- Отображает температуру в виде графика и таблицы

## Технологии
- Backend: Java, Spark
- Кэширование: Redis
- Фронтенд: HTML, CSS, JavaScript

## Запуск проекта
1. Установка Redis
- Скачать Redis с https://github.com/microsoftarchive/redis/releases
- Запустить redis-server.exe

2. Сборка и запуск приложения
- mvn clean compile
- mvn exec:java -D"exec.mainClass=org.example.Main"

3. Использование
- Откройте в браузере: http://localhost:8080
- Введите название города (например: Moscow, London, Tokyo)
- Нажмите "Get Weather" или Enter

## Настройки
- Порт приложения: 8080
- Порт Redis: 6379
- Время жизни кэша: 15 минут
- Количество часов прогноза: 24

# File Upload Service
Простой сервис для загрузки файлов с временными ссылками для скачивания. Позволяет пользователям загружать файлы и получать уникальные ссылки для их скачивания.

## Возможности
- Загрузка файлов
- Временные ссылки
- Авторизация/регистрация
- Персональная статистика
- Автоочистка

## Технологии

- Backend
Java, Встроенный HTTP сервер

- Frontend
Vanilla JavaScript, HTML & CSS

## Установка и запуск

- mvn clean compile
- mvn exec:java -D"exec.mainClass=org.example.Main"
- Откройте в браузере: http://localhost:3000

## Использование

1. Регистрация и вход
- Зарегистрируйте нового пользователя или войдите существующим
- Без авторизации загрузка файлов недоступна

2. Загрузка файлов
- Нажмите на область загрузки или перетащите файл
- Файл автоматически загрузится на сервер
- Появится ссылка для скачивания

3. Скачивание файлов
- Используйте предоставленную ссылку
- Ссылки активны 30 дней с момента последнего скачивания

4. Просмотр статистики
- Видно количество своих файлов и скачиваний
- Список всех загруженных файлов
- История скачиваний
